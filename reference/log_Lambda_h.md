# Logarithm of the integral of the ETAS triggering function

Logarithm of the integral of the ETAS triggering function

## Usage

``` r
log_Lambda_h(theta, th, mh, M0, T1, T2)
```

## Arguments

- theta:

  ETAS parameters `data.frame(mu=mu, K=K, alpha=alpha, c=c, p=p)`.

- th:

  Time of parent event.

- mh:

  Magnitude of parent event

- M0:

  Minimum magnitude threshold

- T1:

  Start of temporal model domain.

- T2:

  End of temporal model domain.

## Value

Logarithm of the integral of the ETAS triggering function
